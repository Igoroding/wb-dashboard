⚙️ Настройка прокси n8n → WB API
1
Импортировать workflow в n8n
Скачайте wb-ads-proxy-workflow.json и импортируйте:
n8n → Workflows → ⋮ → Import from file
Воркфлоу создаёт: POST https://wbtrenz.app.n8n.cloud/webhook/wb-ads-proxy
2
Добавить токены WB в узел «WB API Router»
Откройте узел WB API Router (Code node) в воркфлоу и заполните 4 строки:

TOKENS.oding = токен Одинг, категория Продвижение (read)
TOKENS.stolnikov = токен Стольников, категория Продвижение (read)
STATS_TOKENS.oding = токен Одинг, категория Статистика (read)
STATS_TOKENS.stolnikov = токен Стольников, категория Статистика (read)

Если у вас один токен с двумя категориями — подставьте его в обе переменные.
3
Активировать воркфлоу
Toggle Active в правом верхнем углу воркфлоу.
Затем нажмите «🔌 Тест соединения» ниже.
